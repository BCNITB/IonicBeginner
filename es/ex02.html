Ejercicio 2

En este segundo ejercicio heremos una pequeña aplicacióm donde el jugador tendrá que calcular el resultado de una operación matemática con dos números que se generaran de forma aleatoria por la aplicación.

Se creará una aplicación con el nombre guessOperation. El esqueleto de la aplicación se puede descarregar des del repositorio de bcnitb. En caso de que se quiera ver paso a paso cómo crear este esqueleto y todos los pasos hasta este punto, se recomienda la siguiente página web: Hacer la aplicación paso a paso.

Describiendo la aplicación

La aplicación que acabemos de descargar es el esqueleto de la futura aplicación. Para ver el aspecto actual de la aplicación abriremos un terminal i escribiremos el siguiente comando:

$ionic serve

Tras unos minutos se obrirá el navegador de internet que tengamos por defecto y se mostrará la aplicación como si fuera una aplicación web.

En caso de que querer verse la experiencia en un dispositivo móvil pulsaremos las teclas Control + C y escribiremos el siguiente comando:

$ionic serve -l

Se volverá a abrir un navegador de internet pero veremos la aplicación dentro de un terminal iOS i un Android.

Conceptos

Si se abre el archivo home.page.html, de la carpeta src/app/home del proyecto, se puede observar el código en HTML de la vista. En este código se puede ver que hay una etiqueta de encabezado <h2>, una de párrafo <p> y ptra de formulario <form>, básicamente. Y dentro la etiqueta de formulario hay una <label> y una de un campo de edición <ion-input>.

Ejercicio 3

En este tercer ejercicio trabajaremos con código de TypeScript para retocar la lógica de la aplicación para obtener una respuesta u otra dependiendo de la respuesta del usuario. La lógica de la aplicación está en el fichero home.page.ts que se puede localizar en la misma carpeta que el fichero que hemos visto en el ejercicio 2, b>src/app/home.

Para poder hecer el ejercicio deberemos descarregar la carpeta del ejercicio 3 del repositorio de bcnitb.

Descripción de la aplicación

Si ejecutamos la aplicación veremoa que inicialmente no hay ninguna modificación, pero si introducimos un valor y pulsamos el botón Comprova obtenemos una u otra respuesta depenendiendo de si nuestro cálculo es o no correcte. Si introducimos el resultado correcto obtenemos una confirmación en la pantalla tal y com se muestra en la Resposta correcte. Pero si el resultado que introducimos es inferior al correcto obtendremos el mensaje de T'has quedat curt!".

Conceptos

Ahora mismo, como ya hemos visto, si introducimos un valor al pulsar el botón de comprobación se muestra un mensaje sobre si el valor introducido es correcto o no.

La lógica de este procedimiento se puede observar en el fichero home.page.ts que está en la carpeta src/app/home. Si abrimos este fichero y miramos la función checkAnswer() veremos que esta está formada por un bucle if - else if.

Dentro de un bucle if (si) se entra qcuendo se cumple la condición que hay entre paréntesis. Por ejemplo, el primer bucle if dice:

if(this.sum == this.num){ this.result=this.resultArray[0]; }

A las líneas superiores se está diciendo: if (si) el valor de la variable sum (suma de los números es igual (==) al valor de la variable num (valor introducido) entonces la variable result toma el primer valor de l’array resultArray.

La siguiente condición que se muestra else if indica que en caso de que la primera condición no es cumpla se revise la segunda; pero que si la primera se cumple no se revise la segunda.

Sobre loss array comentar que el primer elemento ocupa la posición 0 del array, ya que en programación se tiene en cuenta el número 0. La definición del array se puede observar donde están declaradas todas las variables.

resultArray:string[]=["Respuesta correcta", "T'has quedat curt!"];

Ejercicio

Si ejecutemos la aplicación y introducimos un valor superior al valor de la suma no se retorna ningún mensaje. El ejercicio consiste en retocar la variable resultArray y la función checkAnswer( ) para que se fecuelva el mensaje: T’has passat de llarg!. Para esto hará falta se retoque el código con el texto comentado TODO.

Un vez finalizado, si ejecutamos la aplicación, la respuessta en pantalla debería tener el texto T'has passat de llarg!.

Ejercicio 1

El primer ejercicio será la instalación de nuestro entorno de trebajo para poder crear aplicaciones con el framework Ionic . Este framework permite la creación de aplicaciones para dispositivos Android, iOS, Windows y navegadores al tractarse de un framework multiplataforma.

Aunque se trata de un framework multiplataforma puede incorporar código nativo de los tres sistemas operativos con relativa facilidad para garantizar que la aplicación se podrá ejecutar en cada todos los terminal.

Para crear el entorn idoneo para poder realizar aplicaciones con Ioni hemos de instalar algunas aplicaciones.

NodeJS y npm

Lo primer que deberemos hacer será instalar las aplicaciones NodeJS y NPM. Para hacerlo iremos a la página web de NodeJS y descargaremos la opción recomanada. Un vez descargada la aplicación ejecutaremos el instalador.

Cuando tengamos la aplicación instalada abriremos un terminal y escribiremos lo siguiente:

$node -v

Al hacerlo se devolveá la versión de NodeJS y nos aseguraremos de que la instalación se ha realizado de forma incorrecta y sin nigún tipos de problema.

Ionic

Para instalar Ionic abriremos el terminal e introduciremos el siguiente comando:

$npm install –g @ionic/cli

La instalación de Ionic durará unos minutos durante los que se irá mostrando todos los paquetes que se están instalanndo an nuestra equipo. Cuando la instalación finalice escribiremos el siguiente comando para ver si todo se ha instalado de forma correcta en nuestro equipo.

$ionic –v

Se nos devolverá la versión de Ionic por línea de comando. En caso contrario querrá decir que la instalación no se ha realizado de forma correcta.

Visual Code

Para facilitar la creación del código es recomendable instalar un editor de texto. En este caso se ha escogido el IDE Visual Code al tener insertado un terminal que permite ejecutar ordenes de comandos para ir escalando y desarrollando las aplicaciones.

Este IDE es gratuito y se puede descargar des de la su página web; tampoco hace falta nos registremos para poder utilizarlo. En caso de que se quiera, podemos instalar cualquier otro IDE como podría ser Atom o un editor de texto. En este último caso se recomienda el editor Notepad++ que también es gratuto.

Crear una carpeta

Un vez instalado todo el entorno crearemos una carpeta en nuestre unidad que contendrá todas las aplicaciones que vamos desarrollando. Se recomienda la creación de esta carpeta para tener todos nuestres proyectes localizados. La creación de la carpeta podemos hacerla des de el explorador de Windows o des de un terminal con el seguiente comando:

$mkdir nom-carpeta

Creando una aplicación

Para crear nuestra primera aplicación abriremos un terminal y nos posiciocionaremos en la carpeta que acabemos de crear y donde tendremos todas las aplicaciones que crearemos. Una vez en la carpeta escribiremos/:

$ionic start firstApp blank --type=angular

Este comando nos creará un esqueleto vacñio de una aplicacióN que hemos nombrado firstApp. Una vez creada la estructura de la aplicación escribiremos,

$cd firstApp

Para acceder a la carpeta de la aplicación. Una vez dentro escribiremos:

$code .

Este último comando nos abrirá nuestro proyecto en Visual Codi. Esta primera aplicación se puede descargar de forma gratuita des del repositorio de github de bcnitb.
